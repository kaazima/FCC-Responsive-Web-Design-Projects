CSS

Introduction

  • What is CSS?

    Like HTML, CSS is not really a programming language. It is not a mark-up language either ! CSS is a stylesheet language.

    • CSS stands for Cascading Style Sheets.
    • It describes how HTML elements are to be displayed on screen, paper, or in other media.

  • Why use CSS?

    CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

    HTML was created to describe the content of a web page. When tags like font, and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process.

    To solve this problem, the World Wide Web Consortium (W3C) created CSS. CSS removed the style formatting from the HTML page!

    Thus, CSS Saves a Lot of Work!

  • CSS Syntax

    A CSS rule-set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. A CSS declaration always ends with a semicolon, and declaration blocks are surrounded by curly braces.

    p {
    color: red;
    text-align: center;
    }

    In the above example,

    p is a selector in CSS.
    color is a property, and red is the property value.
    text-align is a property, and center is the property value.

  • CSS Comments

    Comments are used to explain the code, and may help when you edit the source code at a later date. Comments are ignored by browsers. A CSS comment starts with /* and ends with */

    /*This is a single line comment*/
    p {
    color: red;
    }

    /*This is
    a multi-line comment*/
    h1 {
    color:blue;
    }

CSS Colors

  • Background Color

    The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin).

    Different property values that can be used for background-color are :

    • color
    • transparent
    • initial
    • inherit

    Note: Use a background color that makes the text easy to read.

  • Text Color

    The color property is used to set the color of the text. The default text color for a page is defined in the body selector.

    h1 {
    /*Background Color*/
    background-color: black;

    /*Text Color*/
    color: white;
    }

  • Border Color

    The border-color property sets the color of an element's four borders. An element must have borders before you can change the color. This property can have one to four values.

    /*Top, right, bottom and left borders are set to coral*/

    div {
    border-color: coral;
    }

    /*top,bottom borders are set to blue & right,left borders are set to black*/

    h3 {
    border-color: blue black;
    }

  • Color Values

    Colors in CSS can be specified by the following methods:

    • Hexadecimal colors: A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

    • RGB colors: An RGB color value is specified with the rgb() function. Each parameter (red, green, and blue) defines the intensity of the color and can be an integer between 0 and 255 or a percentage value (from 0% to 100%).

    • RGBA colors: RGBA color values are an extension of RGB color values with an alpha channel, which specifies the opacity of the object. The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

    • HSL colors: HSL stands for hue, saturation, and lightness and represents a cylindrical-coordinate representation of colors. Hue is a degree on the color wheel (from 0 to 360) - 0 (or 360) is red, 120 is green, 240 is blue. Saturation is a percentage value; 0% means a shade of gray and 100% is the full color. Lightness is also a percentage; 0% is black, 100% is white.

    • HSLA colors: HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of the object.

    • Predefined color names: 140 color names are predefined in the HTML and CSS color specification. A list of all predefined names can be found here.

CSS Backgrounds

  • Background color

    The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin).

    Note: Use a background color that makes the text easy to read.

  • Background image

    The background-image property specifies an image to use as the background of an element. By default, the image is repeated so it covers the entire element.

    body {
    background-image: url("paper.jpg");
    }

  • Background repeat

    By default, the background-image property repeats an image both horizontally and vertically. Some images should be repeated only horizontally or vertically, or they will look strange.

    • To repeat an image vertically, set "background-repeat: repeat-y;"
    • To repeat an image horizontally, set "background-repeat: repeat-x;"

  • Opacity

    The opacity property specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0. Lower the value, higher the transparency.

CSS Borders

  • Border Color

    The CSS border properties allow you to specify the style, width, and color of an element's border.

    The border-color property sets the color of an element's four borders. An element must have borders before you can change the color. This property can have one to four values.

    /*Top, right, bottom and left borders are set to yellow*/

    div {
    border-color: yellow;
    }

    /*top border is set to blue, right,left borders to green & bottom border to red*/

    h3 {
    border-color: blue green red;
    }

  • Border Style

    The border-style property specifies what kind of border to display. It can have one to four values (for the top border, right border, bottom border, and the left border). The values can be dotted, dashed, solid, double, groove, ridge, inset, outset, none and hidden.

    Note: None of the other CSS border properties will have ANY effect unless the border-style property is set!

  • Rounded Borders

    The border-radius property is used to add rounded borders to an element:

    p {
    border: 2px solid black;
    border-radius:6px;
    }

CSS Text

  • Text Color

    The color property is used to set the color of the text. The default text color for a page is defined in the body selector.

    h1 {
    color: white; /*Text Color*/
    }

  • Text Decoration

    The text-decoration property is used to set or remove decorations from text. The value "text-decoration: none;" is often used to remove underlines from links.

    The other text-decoration values that are used to decorate text are:

    • overline
    • line-through
    • underline

    Note: It is not recommended to underline text that is not a link, as this often confuses the reader.

  • Text Transformation

    The text-transform property is used to specify uppercase and lowercase letters in a text. It can be used to turn everything into uppercase or lowercase letters, or capitalize the first letter of each word

    p.uppercase {
    text-transform: uppercase;
    }

  • Letter Spacing

    The letter-spacing property is used to specify the space between the characters in a text.

  • Line Height

    The line-height property is used to specify the space between lines.

    p.big {
    line-height: 1.8;
    }

CSS Fonts

  • Font Family

    In CSS, there are two types of font family names:

    • Generic family :- A group of font families with a similar look.
    • Font family :- A specific font family

    The font family of a text is set with the font-family property.It should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font, and so on.

    Note: If the name of a font family is more than one word, it must be in quotation marks, like "Times New Roman".

  • Font Weight

    The font-weight property sets how thick or thin characters in text should be displayed. It specifies the weight of a font.

    The possible property values are:

    • normal
    • bold
    • initial
    • inherit
    • 100-900 (Defines from thin to thick characters)

  • Font Style

    The font-style property is mostly used to specify italic text. The three property values are normal, italic and oblique (Here, the text is leaning).

    h3 {
    font-style: italic;
    }

  • Font Size

    The font-size property sets the size of the text. Being able to manage the text size is important in web design. However, you should not use font size adjustments to make paragraphs look like headings, or headings look like paragraphs. The font-size can be an absolute, or relative value.

    • Setting the text size with pixels gives you full control over the text size.

    • To allow users to resize the text (in the browser menu), many developers use em instead of pixels. 1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.

    • The solution that works in all browsers is to use a combination of percent and em, i.e., to set a default font-size in percent for the body element.

    h1 {
    font-size: 10vw; /*1vw = 1% of viewport width*/
    }
